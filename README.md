# Expense Tracker

A simple Python application to record and track daily expenses.

## Features

- Add new expenses with name, amount, and date
- Store data in a CSV file
- Generate Excel reports using pandas
- Simple command-line interface

## Technologies Used

- Python
- pandas
- CSV
- Excel

## How to Run

1. Install required libraries:

pip install pandas openpyxl

2. Run the program:

python main.py

## Output

The program creates:

- expenses.csv (stores all expenses)
- expense_report.xlsx (summary report)
## Purpose

This project demonstrates basic data handling, file storage, and data analysis using Python and pandas.
